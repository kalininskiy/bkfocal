# FOCAL for BK-0010 (bkfocal)

Расширение VS Code с подсветкой синтаксиса и snippets для языка **Фокал** (FOCAL для семейства БК).

## Возможности

- Подсветка номеров строк, команд, функций и операторов
- Базовые snippets для типичных конструкций
- Поддерживаемые расширения: `.foc`, `.focal`, `.bkf`

## Установка

1. Откройте **Extensions** в VS Code
2. Найдите **BK FOCAL**
3. Установите

## Пример snippets

- `prog` — шаблон программы
- `for` — цикл FOR
- `if` — условный оператор
- `type` — вывод на экран
- `do` — вызов подпрограммы

## Для БК-разработчиков

Рекомендуется использовать вместе с расширением bkbasic - **Vilnius BASIC 86** и расширением bk-ext-tools - инструментами для сборки/запуска в эмуляторе.
